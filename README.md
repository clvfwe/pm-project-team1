# 디미 성적 관리 시스템 (DIMI Score Management System)

학생들의 과목별 성적을 입력받아 병합 정렬 Merge Sort 알고리즘을 통해 학급 석차 및 등급을 계산하고, 반 전체의 주요 성적 지표(평균, 표준편차 등)를 한눈에 확인할 수 있는 C언어 기반 콘솔 프로그램입니다.

---

## Getting Started

이 프로젝트를 개인 컴퓨터 로컬 환경에서 다운로드하고 실행할 수 있습니다.

### Prerequisites 사전 준비 사항

이 프로그램을 실행하고 수정하려면 C언어 컴파일러가 필요합니다.

### Installing 설치 및 실행 방법

1. 이 저장소를 복사하거나 소스 코드 파일을 다운로드합니다.
2. 터미널 또는 커맨드 창을 열고 파일이 있는 폴더로 이동합니다.
3. 다음 명령어를 입력하여 컴파일을 진행합니다:
   gcc main.c -o DimiScoreSystem
4. 생성된 실행 파일을 실행합니다:
   * Windows: DimiScoreSystem.exe 실행
   * Mac/Linux: ./DimiScoreSystem 입력

---

## 주요 기능 및 데모

프로그램을 실행하면 UI 메뉴가 나타나며 다음 기능들을 제공합니다.

1. 성적 등록 및 수정 (update): 반과 번호를 입력한 뒤 7개 과목의 점수를 입력하거나 수정합니다.
2. 학급 석차 계산 (compute_rank): 병합 정렬을 기반으로 특정 과목의 점수를 내림차순 정렬하여 반 석차를 매깁니다.
3. 등급 계산 (compute_grade): 산출된 석차를 기준으로 학생의 최종 등급을 계산합니다.
4. 학급 성적 지표 산출 (print_index): 반 전체의 평균, 등급컷, 그리고 double 자료형과 제곱근 연산을 활용한 정밀한 표준편차를 출력합니다.

## Running the Tests 오류 검사 및 예외 처리

이 시스템은 사용자가 잘못된 값을 입력했을 때 프로그램이 멈추는 것을 방지하기 위해 꼼꼼한 예외 처리 테스트를 거쳤습니다.

### 1. 입력 값 예외 처리 테스트
* 테스트 내용: 점수 입력란이나 메뉴 선택란에 숫자가 아닌 문자(예: 'abc', '한글')를 입력하는 경우
* 결과: 입력 버퍼를 초기화하는 예외 처리 코드(while(getchar() != '\n');)가 작동하여 무한 루프에 빠지지 않고 "올바른 숫자를 입력하세요"라는 안내와 함께 다시 입력을 받습니다.

### 2. 경계선 데이터 테스트
* 테스트 내용: 점수의 최솟값인 0점과 최댓값인 100점을 입력했을 때 석차와 평균, 표준편차가 수학적 오류 없이 정확하게 계산되는지 확인합니다.
